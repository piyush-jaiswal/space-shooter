# Space-Shooter
A simple 2D space shooting game developed using Unity. Deployed using WEBGL.

### Controls
Move - `WASD` / `Arrow keys`
<br>
Shoot - `Left click` / `left CTRL`
<br>
Restart - `R`
<br>
